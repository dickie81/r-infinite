## The Wilson-line correspondence

Under Option 2 / Reading G, the cascade has gauge structure on each
descent path. For the Abelian case (single $U(1)$ generated by the
cascade's complex structure $J$):

**Connection 1-form.** $A(d) = p(d)$, the cascade slicing potential
at layer $d$. Cascade-internal: derived from the digamma function
reflecting log sphere-area decay
($p(d) = \tfrac{d}{dd}\log\Omega_d$ up to constants).

**Wilson holonomy along path $\gamma = [d_a, d_b]$**:
$$W_\gamma = \mathcal{P}\exp\left(\int_\gamma A\right) = \exp\left(\sum_{d \in \gamma} p(d)\right) = \exp\bigl(\Phi(d_b) - \Phi(d_a-1)\bigr).$$

**Cascade running coupling formula**: For a gauge-anchored
observable at layer $d_B \in \{12, 13, 14\}$, Part IVa Theorem
`forced-paths` gives
$$\log(Q_{\rm obs}/Q_{\rm bare}) = \Phi(d_B) = \sum_{d=5}^{d_B} p(d).$$

This is exactly $\log W_{[5, d_B]}$ — the log Abelian Wilson
holonomy along the cascade descent from $d=5$ to $d_B$.

So the cascade's existing formula IS a Wilson line. The
re-interpretation is exact, not approximate.

## Cascade-internal-ness of the connection

The connection 1-form $A(d) = p(d)$ is genuinely cascade-internal:

1. $p(d)$ is defined in Part IVa Definition `cascade-potential` from
   the digamma function $\psi((d+1)/2)$.

2. Digamma is the log derivative of Gamma:
   $\psi(x) = \tfrac{d}{dx}\log\Gamma(x)$.

3. Sphere area $\Omega_d = 2\pi^{(d+1)/2}/\Gamma((d+1)/2)$ has
   $\log\Omega_d = \log(2) + ((d+1)/2)\log\pi - \log\Gamma((d+1)/2)$,
   so
   $\tfrac{d}{dd}\log\Omega_d = \tfrac{1}{2}\log\pi - \tfrac{1}{2}\psi((d+1)/2) = -p(d)$.

4. Therefore $A(d) = p(d) = -\tfrac{d}{dd}\log\Omega_d$ is the
   *negative log decay rate of sphere areas*.

The connection is the cascade's slicing-rate-per-layer, expressed
as a 1-form on the cascade descent path.

## Why this is genuinely a Wilson line, not a mere relabelling

Three structural features make the identification non-trivial:

**(a) Path-ordering is automatic.** The cascade's discrete layers
have a natural total order (descending from large $d$ to small $d$),
so the path-ordered exponential $\mathcal{P}\exp(\int A)$ is just the
ordinary product of layer factors. In a non-Abelian extension this
matters; in the Abelian case it's automatic.

**(b) Gauge invariance is consistent.** A gauge transformation
$\psi_d \to e^{i\phi_d}\psi_d$ at each layer transforms the connection
$A(d) \to A(d) + (\phi_d - \phi_{d-1})/\Delta d$. Under this, the
Wilson line transforms as $W \to e^{i(\phi_b - \phi_a)} W$, so
$|W|^2$ is gauge-invariant. The cascade's $|\exp(\Phi)|^2$ is the
gauge-invariant content.

**(c) Composition is functorial.** Two paths concatenated give
$W_{\gamma_1 \cdot \gamma_2} = W_{\gamma_2} \cdot W_{\gamma_1}$
(in path order). The cascade's potential $\Phi$ satisfies
$\Phi(d_3) - \Phi(d_1) = (\Phi(d_3) - \Phi(d_2)) + (\Phi(d_2) - \Phi(d_1))$,
so the Wilson line composes correctly.

These are properties any genuine Wilson line must have; the cascade's
existing formula has them all.

## Non-Abelian extension: open

For the full $SU(3)$ Wilson line at gauge layer $d=12$:

- Adams' theorem: $\rho(12) - 1 = 3$ linearly independent vector
  fields on $S^{11}$.
- These 3 vector fields generate a 3-dim subspace of the tangent
  space at each point of $S^{11}$.
- For an $SU(3)$ structure, we need these to form a basis for an
  $\mathfrak{su}(3)$-valued 1-form (Lie algebra valued connection).

What's missing cascade-internally:

- An identification of the 3 vector fields with specific generators
  of $\mathfrak{su}(3)$ (e.g., Gell-Mann matrices in some basis).
- A construction of the connection 1-form valued in $\mathfrak{su}(3)$
  along the cascade path.
- Curvature and field strength tensors derived from cascade structure.
- Non-perturbative gauge dynamics (gluon self-coupling, confinement,
  instantons).

These are the steps that would lift the cascade from a perturbative
running-coupling theory to a full non-perturbative gauge theory. The
cascade currently provides the perturbative running; the
non-perturbative completion is open.

**This is significant.** It means Option 2 / Reading G recovers the
cascade's existing perturbative gauge structure but does not yet
deliver QCD's non-perturbative content (asymptotic freedom is
visible in the running, but confinement is not derived). The
cascade's claim to "reproduce the Standard Model" is delivered for
running couplings, not for non-perturbative gauge dynamics.
